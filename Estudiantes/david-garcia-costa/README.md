# TFG - David García Costa

Este proyecto desarrolla un prototipo de apoyo al proceso de QA que permite generar y gestionar casos de prueba a partir de documentación funcional, combinando una aplicación web manual con agentes de IA y publicación en Kiwi TCMS.
